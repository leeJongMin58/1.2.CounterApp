숫자세기(계수기 앱) 

- LinearLayout을 이용하여 화면 그리기
- 유저 인터페이스구성

---

step1. 개요

- 간단한 기능을 구현하고 , UI를 그릴 수 있다.
    
    ![계수기.jpeg](https://prod-files-secure.s3.us-west-2.amazonaws.com/ceda91ce-02d3-476b-9c9a-f1c5beb70a43/d9046dcd-347c-44db-8db9-598517e98144/%E1%84%80%E1%85%A8%E1%84%89%E1%85%AE%E1%84%80%E1%85%B5.jpeg)
    
- 버튼을 누르면 숫자가 1씩 올라감
- 초기화 기능이 있음
- 9,999까지 숫자를 셀수 있음
- 고리가 있어 손에 고정하기가 펴함

---

구현기능

- + 버튼을 클릭시, 숫자 1씩 올리기
- 초기화 버튼을 클릭시 숫자를 0으로 변경하기
